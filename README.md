さらに、サブスク確認のリマインドが月末に来るようにした。
（以下、前回作ったものの概要）
PHP, Mysqlを使って、ユーザー登録/ログイン機能、データベースにデータを保存する機能を実装。
今回は、色々なパスワードをデータベースを管理するという立て付け。
